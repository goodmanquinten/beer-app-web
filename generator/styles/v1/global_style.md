# Global Style — v1

- Clean cartoon vector illustration with rich cel-shading — NOT painterly, NOT realistic
- Smooth flat color fills with distinct shading bands — no brush strokes, no paint texture, no canvas grain
- Confident clean outlines, slightly varying in weight (thicker on silhouette, thinner on details)
- Slightly stocky/chunky proportions — stylized but polished
- COLOR FIDELITY: ONLY use colors that appear in the reference image. Do NOT add gold, metallic, or any colors not present in the original label. If the reference has silver trim, the render has silver trim. If the reference has white text, the render has white text. No embellishment.
- Do NOT add decorative elements that are not in the reference — no extra borders, flourishes, ribbons, or accents that aren't on the real label
- NO background — fully transparent layer
- Container centered in frame
- Can lid and bottom rim: ALWAYS metallic silver/gray
- NO drop shadow — the container floats on a transparent background with no shadow at all
- Simplify fine print into decorative lines — only main brand name needs to be legible
- The ENTIRE container must be visible — do NOT crop or cut off the top (lid) or bottom (base) of the container. Leave generous padding on ALL sides.
