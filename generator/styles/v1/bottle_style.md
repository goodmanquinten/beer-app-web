# Bottle Style — v1

- Standard brown or green longneck beer bottle proportions (12 oz / 355 ml), depending on the reference
- Slight 3/4 front angle, matching the can renders so the shelf feels consistent
- Clean cartoon vector illustration with the same bold outline and cel-shaded treatment as the can style
- Glass reads as glossy but simplified: 3-4 distinct shading bands, no realistic caustics or photographic reflections
- Neck, shoulders, and body must feel like a single intentional silhouette — polished, chunky, and slightly stylized
- Label area clearly readable with gentle cylindrical wrap, preserving the real label layout and color zones from the reference
- Crown cap visible and simplified, with clean metallic shading only if the real bottle has a visible cap
- COLOR FIDELITY: only use colors present in the reference label and bottle; do not invent extra gold, glow, or decorative accents
- NO drop shadow — no floor shadow, no cast shadow, no shadow crescent beneath the bottle
- Full bottle must be visible from cap to base with generous transparent padding on all sides
